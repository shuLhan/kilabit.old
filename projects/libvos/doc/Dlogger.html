Description

Dlogger is a module for writing formatted string to a file or standard error, or both. What make Dlogger module different with normal File module is it have time-stamp added at the beginning of each line, 

[YEAR.MONTH.DAY HOUR:MINUTE:SECOND] log output

If Dlogger object is not initialized, by calling open() method, all log output will be printed to standard error.
