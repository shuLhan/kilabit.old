Description

DNS_rr is a module for handling content of Resource Record (RR) in DNS packet.

Class Variables

TypeNameDescription
uint16_t _type type of RR object.
uint16_t _class class of RR object.
uint32_t _ttl time to live for RR object.
uint16_t _len length of RR data object.
Buffer _name domain name for RR object with type is MX.
Buffer _data extract value for A record, NS record, SOA MNAME field, PTR record, and HINFO CPU field.
Buffer _data2 extract value for SOA RNAME field and HINFO OS field.
uint32_t _serial SOA serial value.
uint32_t _refresh SOA refresh value.
uint32_t _retry SOA retry value.
uint32_t _expire SOA expire value.
uint32_t _minimum SOA minimum value.
uint16_t _mx_pref priority of RR if type is MX.
static unsigned int RDATA_MAX_SIZE maximum size for RDATA field in RR.
DNS_rr* _next pointer to the next RR object.
