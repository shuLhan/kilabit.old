Description

This module handle list of virtual user account on FTPD module. When using FTP server service, through FTPD module, you need a virtual account to let only authenticated user to log into your service. You must not use your system account when setting up and using a service in open protocol, like FTP.

Class Variables

TypeNameDescription
_name : a user name for the service.
_pass : a plain-text password for authenticating user into the service.
_next : pointer to the next FTPD_user object.

Code static int IS_EXIST(FTPD_user* users, Buffer* name, Buffer* pass = NULL)

name : a user name that will be checked in the list.

Return Value 0 : false, user with 'name' and/or 'pass' is not exist in the list.

Description Check for user name with name is name in the list of user list. If pass is not nil, also check for password if it's equal or not.
