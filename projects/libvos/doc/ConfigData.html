Description

ConfigData is a class that mapping all header, key, value, and comment from config file to the list of object in memory.

Class Variables

TypeNameDescription
int _t type of object, valid values:
  • CONFIG_T_NONE : (0) object is unset.
  • CONFIG_T_HEAD : (1) object is a header.
  • CONFIG_T_KEY : (2) object is a key.
  • CONFIG_T_VALUE : (3) object is a value of key.
  • CONFIG_T_MISC : (4) object is a comments.
ConfigData* _value pointer to a config value.
ConfigData* _next_head pointer to the next config header in the list.
ConfigData* _last_head pointer to the last config header in the list.
ConfigData* _next_key pointer to the next config key in the list.
ConfigData* _last_key pointer to the last config key in the list.
