WUI

A Complete Control Web CMS.

The idea behind WUI is simple:

  • You have a html file and you want to publish it to the internet.
  • You want to group your html files by menu-driven interface.
  • Let the user (browser) do the process, server only provided the data and the script.
  • Database-less, running without need of database service.
  • Complete control of your site.

WUI is a hack to HTML meta tag, HTML DOM, and JavaScript to create a semi-dynamic website.

docit

A Document generator for C/C++ code.

docit parsing a formatted comment on source code files and generate a HTML files for easy viewing. The primary goal is to create an easy way for documenting an API/libraries/program while not burdening developer with separate files for documenting the module, class, method, and attributes; and with a clear syntax without polluting the source codes.
